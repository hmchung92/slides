RequireJS

  • JavaScript file and module loader
  • Loader for AMD (Asynchronous Module Definition)
  • Visit: http://requirejs.org

Examples

  • Sample 1 - examples/requirejs/basic/
  • Sample 2 - examples/requirejs/widget/
  • Sample 3 - examples/requirejs/app/

Run build

  1. Go to the app example (example 3) using a command-line terminal
  2. Run: node js/lib/r.js -o build.js
  3. r.js is the optimizer, build.js is the build configuration file
